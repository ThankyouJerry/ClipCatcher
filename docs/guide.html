Quick Guide

기본 사용 흐름

링크를 넣고, 정보를 가져오고, 범위와 화질을 정한 뒤 다운로드합니다.

1

URL 입력

치지직 VOD/클립 또는 YouTube 링크를 입력합니다.

2

정보 가져오기

영상 제목, 썸네일, 사용 가능한 화질을 확인합니다.

3

다운로드 범위 선택

전체 다운로드 또는 시간 범위 지정 다운로드를 선택합니다. 시간은 90처럼 초 단위로도, 01:30:00처럼 시:분:초 형식으로도 입력할 수 있습니다.

4

다운로드 센터 확인

진행중, 완료, 실패 탭에서 각 작업 상태를 확인합니다.

Time Range Example

구간 다운로드 입력 예시

시작: 5:00:00
종료: 6:00:00

또는

시작: 18000
종료: 21600

긴 VOD는 먼저 작은 구간으로 테스트한 뒤 필요한 범위를 넓히는 것을 추천합니다.
