# 사용 예시 (초안)

## 기본 다운로드
1. URL 입력란에 치지직 VOD/클립 또는 YouTube 링크를 입력합니다.
2. `정보 가져오기`를 눌러 메타데이터를 확인합니다.
3. 원하는 화질을 선택하고 `다운로드`를 누릅니다.
4. 우측 `다운로드 센터`에서 진행 상태를 확인합니다.

## 구간 다운로드
1. `다운로드 범위`에서 `시간 범위 지정 다운로드`를 선택합니다.
2. 시작/종료 시간을 입력합니다.  
   예: `5:00:00` ~ `6:00:00` 또는 초 단위 `18000` ~ `21600`
3. `다운로드`를 누르면 지정 구간만 저장됩니다.

## 다운로드 센터 탭
- `진행중`: 현재 실행/대기 중인 항목 확인
- `완료`: 성공적으로 저장된 항목 확인 및 파일 열기
- `실패`: 오류 메시지 확인 후 재시도 대상 점검

## 팁
- 기본 설정은 순차 다운로드(동시 1개)입니다.
- 설정에서 동시 작업 수를 늘리면 병렬 다운로드도 가능합니다.
- `yt-dlp`, `ffmpeg`가 없으면 앱 시작 시 설치 안내 팝업이 표시됩니다.
