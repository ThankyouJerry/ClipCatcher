Before Download

저작권 안내

ClipCatcher는 사용자가 접근 가능한 영상의 저장을 돕는 도구입니다. 프로그램 사용과 다운로드는 자유롭게 할 수 있지만, 영상과 음성 등 콘텐츠의 저작권은 원 저작권자에게 있습니다. 개인 보관 범위를 넘어 재업로드, 공유, 편집물 공개, 상업적 이용을 할 때는 각 플랫폼의 이용약관과 원 저작권자의 허락 범위를 반드시 확인해주세요.

Source

소스에서 실행

git clone https://github.com/ThankyouJerry/ClipCatcher.git
cd ClipCatcher
pip install -r requirements.txt
python src/main.py

다운로드가 실패하면 필수 도구 설치 안내와 실패 탭의 오류 메시지를 먼저 확인하세요.
