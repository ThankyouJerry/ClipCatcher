Source

소스에서 실행

git clone https://github.com/ThankyouJerry/ClipCatcher.git
cd ClipCatcher
pip install -r requirements.txt
python src/main.py

다운로드가 실패하면 필수 도구 설치 안내와 실패 탭의 오류 메시지를 먼저 확인하세요.
