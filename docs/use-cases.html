Real Use Cases

이럴 때 ClipCatcher를 씁니다.

웹에서 분석을 대신 해주는 서비스처럼 보이기보다, 실제로 편집자가 마주하는 다운로드 상황을 기준으로 보여주는 페이지입니다.

01

긴 방송 중 특정 1시간만 저장

전체 VOD가 8시간이어도 시작/종료 시간을 지정해 필요한 구간만 받습니다. 예를 들어 5:00:00부터 6:00:00까지만 저장할 수 있습니다.

시간 범위 지정 HH:MM:SS 입력 구간 다운로드
02

치지직 클립과 VOD를 한 앱에서 관리

치지직 VOD, 치지직 클립, YouTube 링크를 같은 흐름으로 불러와 메타데이터와 화질을 확인한 뒤 다운로드합니다.

CHZZK VOD CHZZK Clip YouTube
03

여러 링크를 순차적으로 처리

다운로드가 겹쳐 앱이 불안정해지는 것을 피하기 위해 기본 흐름은 순차 다운로드에 맞춰져 있습니다. 진행중/완료/실패 탭으로 상태를 구분합니다.

대기열 완료 탭 실패 탭
04

하이라이트 후보만 받아 편집 준비

분석 도구에서 찾은 후보 시점을 ClipCatcher로 가져와, 핵심 반응 앞뒤 여유를 포함한 추천 컷 범위만 다운로드하는 흐름으로 확장할 수 있습니다.

ClipRadar JSON 추천 컷 편집 준비

Failure Handling

실패한 작업도 따로 확인

!

필수 도구 안내

다운로드와 병합에는 ffmpeg, ffprobe, yt-dlp 같은 도구가 필요할 수 있습니다. 앱은 필요한 도구가 없을 때 설치 안내를 표시합니다.

ClipCatcher는 다운로드 자체를 안정적으로 처리하는 메인 앱입니다.
