## ClipCatcher v2.0.8

### 주요 변경 사항

- 치지직 빠른 다시보기의 API 영상 길이를 참고값으로 표시합니다.
- 시간 범위 다운로드를 실제 HLS 재생목록 길이로 다시 검증합니다.
- 서명된 HLS 주소의 화질별 fallback 처리를 안정화했습니다.
- 게시 시각과 영상 길이를 별도 항목으로 표시해 긴 영상의 범위를 확인하기 쉬워졌습니다.
- YouTube의 Final Cut Pro 호환 `H.264 + AAC` 선택 동작은 그대로 유지합니다.

### 다운로드

- Windows: `ClipCatcher-Windows.zip`
- macOS: `ClipCatcher-macOS.zip`

### 참고

다운로드한 콘텐츠의 저작권은 원 저작권자에게 있으며, 플랫폼 이용약관과 저작권자의 허락 범위를 확인해 주세요.

전체 변경 이력은 [CHANGELOG.md](https://github.com/ThankyouJerry/ClipCatcher/blob/main/CHANGELOG.md)에서 확인할 수 있습니다.
